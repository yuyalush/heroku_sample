== 日経Linux 2012年10月号

Amazonへの商品検索機能が付いた蔵書管理の簡易アプリです。

Herokuへのデプロイも可能になっています。

詳細については、日経Linux 2012年10月号をご確認ください。

このソースは一部不完全になっております。記事を参考に、以下のファイルへ必要事項の追記等を行い利用してください。

=== 編集が必要なファイル

- config/database.yml

ローカルのPostgreSQLに合わせて変更してください。

- config/application.rb

AmazonAWSのアカウントや、アフィリエイトIDの設定をしてください。

- env

envを.envに変更してください。
